Hi, There. I am the feature goat and here are some of the features in our on-line whiteboard application
KEY BINDINGS
Ctrl + z = undo
Ctrl + y = redo
Hold Ctrl + Hold mouse1 = for moving many shapes
Ctil + click = select multiple items
delete button = delete selected item(s)

SAVE/LOAD Files
To save your project use the export feature then copy the text generated to
your favorite text editor and save under a file name of your choice.
This will save all drawings on all your boards andthen later to load
your project just copy the text back from your text file
and hit import. If you must have a picture file we suggest using
snipping tool(windows build-in), print screen or something of that sort.

TEMPLATES
You can make a template of custom shapes that you have created
out of many shapes.
Fyrst make sure you have the select tool active
then Ctil + click to select multiple items
and then hit create template to give it a name.
Now you should be able to use your pre made item on all boards.
